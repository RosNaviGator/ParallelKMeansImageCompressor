Detailed Description

The main program for decoding .kc files generated by the Encoder.

This program provides a command-line interface for decoding images that have been compressed by the Encoder. It allows users to load a compressed image, convert it to a viewable format, and optionally save it as a .jpg file. The program makes use of OpenCV for image processing and the custom filesUtilsand utilsCLI libraries for handling file operations and user interactions. The Decoder program reads a .kc compressed file, decodes it into an image matrix, converts the color space for display, and allows the user to save a decompressed copy as a .jpg image. The program provides a simple command-line interface for selecting files and configuring output options.
