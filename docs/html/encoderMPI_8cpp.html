Detailed Description

Main entry point for the MPI-based image compression application.

This program compresses an image using the K-means clustering algorithm with MPI (Message Passing Interface) for parallel processing across multiple processors. It reads input parameters from the command line, processes the image, applies compression, and saves the compressed image to a binary file. The program also evaluates performance metrics and writes them to a CSV file.

The application uses OpenCV for image processing and custom libraries for compression, file handling, and performance evaluation.
